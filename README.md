# hestiaorg.com

Static public website for HESTIA.

## Public identity policy

HESTIA is presented as an institution rather than as a founder-led or personality-led organisation.

- Do not publish personal names, biographies, photographs or founder credits.
- Use institutional language such as “HESTIA exists to…” and “HESTIA researches…”.
- Use role-based public contact addresses. The principal address is `info@hestiaorg.com`.
- Public pages should preserve, document, research, teach, organise or responsibly share knowledge—not promote individuals.
- This policy does not apply to private legal, regulatory, contractual, financial or domain-registration records where identification is required.

## Files

- `index.html` — page structure and public copy
- `styles.css` — visual system and responsive layout
- `script.js` — mobile navigation and dynamic copyright year
- `assets/hestia-dandelion-master.png` — approved dandelion artwork
- `CNAME` — GitHub Pages custom domain

## Local preview

Open `index.html` in a browser, or run a local static server.

## Deployment

Designed for GitHub Pages from the repository's `main` branch and root directory.

## Before launch

Confirm that `info@hestiaorg.com` is active and able to receive mail.


## Launch revision

The single-page launch site includes the approved “The HESTIA Emblem” text within About HESTIA. No founder or personal identity appears publicly.
